About Donut and Other Information!

Donut's Development

Donut was developed in a variaty of languages to give the best experience to all its users, it was mainly developed in Python and JavaScript, the website was developed with HTML and CSS markup, the bot was developed over months and tested a lot to gurantee satisfaction for all of its users! Python is responsible for the main bot's system and the backend webpage routes making it very vital to the development. On the other hand JavaScript is responsible for a number of things, first, it is responsible for dealing with requests made from the website on the dashboard to change things in servers, second, its responsible for some of the image manipulation within the bot, last, it is responsible for any other frontend functions in the webpage. HTML is used for the actual content on the webpage and CSS is just there to style it!



Why Donut?

Donut isn't developed for a specific purpose, its a multi purpose bot which means that it can be used for pretty much anything, having Donut as the only bot in your server can also work because of how flexible and dynamic it is. Donut's development was also a very smooth process, Donut was developed in multiple languages to give you the best experience while using the bot, it was developed and tested for many months to secure everyone's satisfaction while using the bot in their own server.
